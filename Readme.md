# Pràctica: Gestió d'Excepcions i Validacions en PHP  

**Descripció:**  
Aquesta pràctica se centra en l'ús de **gestió d'errors i excepcions** en PHP, així com en la **validació de formularis**.  
L’objectiu és aprendre a detectar, controlar i comunicar situacions d’error d’una manera segura i ordenada, utilitzant tant **sentències `try-catch`** com **excepcions personalitzades** i **filtres de validació**.

- **Llenguatge:** PHP  
- **Entorn d'execució:** XAMPP
